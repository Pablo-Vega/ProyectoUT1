# PabloPVA

Componente que muestra una tarjeta con un avatar, un título y una descripción.

#### Ejemplo

<Canvas of={PabloPVA.Primary} />

## Uso

```tsx
import PabloPVA from 'milibreria';

function Example() {
  return (
    <PabloPVA
      title="Pablo Vargas"
      description="Frontend Developer Extraordinaire"
      avatar="https://example.com/pablo-avatar.jpg"
    />
  );
}

export default Example;
